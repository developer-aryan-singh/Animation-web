 # Animation Project

## Description
This project demonstrates animations using GSAP (GreenSock Animation Platform) and ScrollTrigger. It integrates Bootstrap for responsive design and jQuery for additional scripting functionality.

## Technologies Used
- **HTML5** - Structure of the webpage
- **CSS3** - Styling and layout
- **Bootstrap 5.3.3** - Responsive design framework
- **GSAP 3.3.2** - JavaScript animation library
- **ScrollTrigger** - GSAP plugin for scroll-based animations
 

## Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project folder:
   ```sh
   cd <project-folder>
   ```
3. Open `index.html` in your browser.

## File Structure
```
project-folder/
│-- assets/
│   │-- css/

│   │-- js/
│   │   ├── main.js
|   │-- font/
```

## Usage
- Ensure an internet connection for CDN-hosted libraries (Bootstrap, GSAP, and jQuery).
- Modify `main.js` to add custom animations.
- Update `style.css` for styling changes.

## Dependencies
The following external libraries are used in this project:
- [Bootstrap 5.3.3](https://getbootstrap.com/)
- [GSAP 3.3.2](https://greensock.com/gsap/)
